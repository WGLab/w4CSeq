w4CSeq
A demo web server to process 4C-Seq data

w4CSeq applies a customized pipeline to deal with both enzyme digestion and sonication fragmentation based 4C-seq data, identifies statistically significant regions, draws interacting plots for intra-chromosome and inter-chromosome interactions, and overlays genomic features (TSS, TTS, CpG sites), DNA replication timing and user-provided annotation onto it. The combined plot will help uncover significant features in/around 4C regions. (Note: This website is a demo server for you to examine, thus may not handle large amounts of data.)
